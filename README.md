📚 Library Management System

A simple and interactive Library Management System built using Python. This is a menu-driven console application that allows users to manage books efficiently using Python lists, functions, loops, and conditional statements.

This project was created as part of my Python learning journey to strengthen my programming fundamentals through real-world projects.

---

✨ Features

- ➕ Add New Books
- 📖 View All Available Books
- 🔍 Search Books by Name
- 🗑️ Remove Books from the Library
- 📚 Display Total Number of Books
- 🚪 Exit the Program

---

🛠️ Technologies Used

- Python 3
- Lists
- Functions
- While Loop
- Conditional Statements ("if", "elif", "else")
- User Input

---

📋 Menu

====== Library Management ======

1. Add Books
2. View Books
3. Search Books
4. Remove Books
5. Total Books
6. Exit

---

▶️ How to Run

1. Download or clone this repository.
2. Make sure Python 3 is installed.
3. Open the project in Python IDLE, VS Code, or any Python IDE.
4. Run:

python library_management.py
